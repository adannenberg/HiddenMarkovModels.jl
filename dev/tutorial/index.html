Tutorial

Work in progress

In the meantime, you can take a look at the files in test, which demonstrate more sophisticated ways to use the package.

julia> using HiddenMarkovModels, Distributions

Constructing an HMM:

julia> function random_gaussian_hmm(N)
           p = ones(N) / N  # initial distribution
           A = rand_trans_mat(N)  # transition matrix
           dists = [Normal(randn(), 1.0) for n in 1:N]  # observation distributions
           return HMM(p, A, dists)
       end;

Checking its contents:

Simulating a sequence:

julia> state_seq, obs_seq = rand(hmm, 1000);
julia> first(state_seq, 10)'1×10 adjoint(::Vector{Int64}) with eltype Int64:

julia> first(obs_seq, 10)'1×10 adjoint(::Vector{Float64}) with eltype Float64:

Computing the loglikelihood of an observation sequence:

julia> logdensityof(hmm, obs_seq)-1447.8625114170436

Inferring the most likely state sequence:

julia> most_likely_state_seq = viterbi(hmm, obs_seq);
julia> first(most_likely_state_seq, 10)'1×10 adjoint(::Vector{Int64}) with eltype Int64:
 3  1  2  1  2  1  2  3  1  2
