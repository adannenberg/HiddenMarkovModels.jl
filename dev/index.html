HiddenMarkovModels.jl

A Julia package for HMM modeling, simulation, inference and learning.

Main features

Performance
  • allocation-free versions of core functions
  • linear algebra subroutines
  • multithreading
  • compatibility with SparseArrays and StaticArrays.jl
Genericity
  • number types are not restricted
  • observations can be arbitrary Julia objects (not just scalars or arrays)
  • emission distributions dist only need to implement
    • rand(rng, dist)
    • logdensityof(dist, x) (following DensityInterface.jl)
Automatic differentiation
  • in forward mode with ForwardDiff.jl
  • (soon) in reverse mode with ChainRules.jl
Reliability
  • same outputs as an independent other package, up to numerical precision
  • quality checks with Aqua.jl
  • correctness and type stability checks with JET.jl
  • documentation with Documenter.jl
  • benchmarks compatible with PkgBenchmark.jl
